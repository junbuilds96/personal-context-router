# Changelog

All notable changes to this project will be documented here.

## 0.1.0 - 2026-05-25

- Sharpened README positioning, quickstart, artifact expectations, and product
  boundaries.
- Added `pcr diagnose` packet diagnostics with Markdown reports and nonzero
  failure exits.
- Added a synthetic agent handoff case study.
- Added contributor, security, and development workflow docs.
- Added a Makefile for verify, smoke, and clean tasks.
- Initial local-first CLI for redaction, extraction, approval, packet creation,
  request artifacts, writebacks, and the synthetic sample workflow.
